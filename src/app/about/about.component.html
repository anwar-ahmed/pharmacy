This application is built using HTML, CSS, TypeScript and AngularJs.


The motive of this application is to provide the details of popular medicines.


Pharmacy Assistant can update or delete existing medicines and add new medicines.


It includes information like Medicine Name, Manufacturer, Batch No, Price, Expiry Date etc.


Pharmacy Assistant can find the medicines of various types such as Syrup, Gel, Tablet, Capsule etc.
